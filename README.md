# Simple video player

# Get started

First, add videos to `/tmp` folder and updates the `server.json` file.

Install json-server
```sh 
npm install -g json-server 
```

Run json-server into folder's project
```sh
json-server --host 127.0.0.1 -p 3000 server.json
```

Then
```sh
yarn install
npm run dev
```
